## 系统架构

```
数据集展示网站
├── 前端界面 (Bootstrap 5 + 自定义CSS/JS)
│   ├── 首页 - 数据集列表和筛选
│   ├── 详情页 - 数据集详细信息
│   ├── 编辑页 - 编辑数据集信息
│   ├── 新建页 - 创建新数据集
│   └── 上传页 - MD文件上传和解析
├── 后端API (Flask + SQLAlchemy)
│   ├── 数据模型 - Dataset, Author, Category
│   ├── 路由处理 - CRUD操作
│   ├── MD解析器 - 自动提取关键字段
│   └── 文件处理 - 上传和解析
└── 数据库 (SQLite)
    ├── 12个数据集记录
    ├── 4个器官分类
    └── 作者关联关系
```

## 维护指南

### 数据备份
```bash
# 备份数据库
cp dataset.db dataset_backup_$(date +%Y%m%d).db
```

### 添加新数据集
1. 通过网页界面上传MD文件
2. 或手动填写数据集信息
3. 或直接将MD文件放入对应分类目录后运行：
```bash
python3 init_database.py --init
```

### 清理测试数据
```bash
python3 init_database.py --clear
python3 init_database.py --init
```

### 常见问题
1. **端口占用**: 修改app.py中的端口号
2. **数据库锁定**: 重启应用释放连接
3. **文件上传失败**: 检查uploads目录权限
